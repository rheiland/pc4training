GUI Overview

  • Terminal tab:   where a user can edit, compile, and run code
  • Out: Plots tab: output display of cells and substrates

Introduction

This app provides a general purpose PhysiCell workbench that lets you compile/run models and visualize results.

This model and cloud-hosted demo are part of a course on computational multicellular systems biology created and taught by Dr. Paul Macklin in the Department of Intelligent Systems Engineering at Indiana University. It is also part of the education and outreach for the IU Engineered nanoBIO Node and the NCI-funded cancer systems biology grant U01CA232137. The models are built using PhysiCell: a C++ framework for multicellular systems biology [1].

Basic instructions

In the "Terminal" tab, go to the directory where your PhysiCell model is located. Compile your model (typically, just build the example models as described in the Quickstart guide) and run it.

In the dropdown directory widget, select the output directory containing your model's results.

Click the "Out: Plots" tab to visualize the results in the output directory. Choose a substrate from the drop-down menu, and slide through the saved times. As the simulation runs, the "max" field (maximum frame number) will increase, so you can view more simulation frames.

Note that you can download output data for further exploration in your tools of choice.
